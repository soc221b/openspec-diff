# Shipping - Changes

## MODIFIED Requirements

### Requirement: The system SHALL present clear shipment progress so a customer can understand where an order is in transit
The system SHALL present shipment progress with more timely tracking details so a customer can follow delivery changes clearly.

#### Scenario: A customer checks delivery progress
- **WHEN** a shipped order is viewed from the order details page
- **THEN** the current shipment progress is shown with fresher detail
