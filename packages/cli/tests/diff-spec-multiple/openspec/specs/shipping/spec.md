# shipping Specification

## Purpose
This specification describes how delivery progress is presented after an order leaves the warehouse.

## Requirements

### Requirement: The system SHALL present clear shipment progress so a customer can understand where an order is in transit
The system SHALL present clear shipment progress so a customer can understand where an order is in transit.

#### Scenario: A customer checks delivery progress
- **WHEN** a shipped order is viewed from the order details page
- **THEN** the current shipment progress is shown clearly
